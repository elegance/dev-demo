这是一个关于公交线路、公交站点的查询语句，如下
> 
line              	station
301               站点1，站点2，站点3，站点4
302               站点4，站点5，站点6，站点7
303              站点2，站点3，站点4，站点8
304               站点9，站点1，站点4，站点6
> 
结果是以站点为单位，得出途经该站点的线路，如：
station             line
站点1                301，304
站点2                301，303
站点3                301，303
站点4                301，302,304
站点5                302
站点6                302,304
站点7                302
站点8                303
站点0                304
