抽象空白问题(white-space)：文本可能溢出指定宽度、文本可能有空格、换行 

        white-space: 处理空白问题(空格、换行)，
            normal: 合并空格，忽略换行，文本超出自动换行
            pre: 保留空格，保留换行，不自动换行
            nowrap: 合并空格，不换行，文本超出不自动换行
            pre-line: 保留换行，合并空格，超出文本自动换行
            pre-wrap: 保留空格，保留换行，超出文本自动换行
